# zencart
Zen Cart® truly is the art of e-commerce; free, user-friendly, open source shopping cart software. The ecommerce web site design program is developed by a group of like-minded shop owners, programmers, designers, and consultants that think ecommerce web design could be, and should be, done differently.
## the changes in the code
Zen Cart® provides basic functionality, but the skin and the layout is not good looking, and need some important modules. Such as database backup, source code backup, SEO and image handlesr. I use v1.15.1 is because there is more useful plugins to use. The fold [Plugins](https://github.com/pangqiao/zencart/tree/master/Plugins) contains the import plugins in the change, including:
